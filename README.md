# UserUI
OnlineAttandance UI

React.js와 Material UI를 활용해서 만드는 프로젝트입니다.

Material UI
https://mui.com/material-ui/getting-started/installation/

Create-React-App
https://create-react-app.dev/

node.js
https://nodejs.org/en/

기능
  1. 오늘의 말씀
  2. 출석체크
  3. 로그인
  4. 회원가입
  5. 개인정보 수정
 
 총 페이지는 오늘의 말씀, 출석체크, 로그인, 회원가입, 개인정보수정 페이지로 구성되어있습니다.
 
이 프로젝트를 Clone해서 사용할때
node js가 설치되어있어야합니다.

1. npm install
  필요한 패키지를 다운받는 명령어
2. npm start
  서버를 실행시키는 명령어
